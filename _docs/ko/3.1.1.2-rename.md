어플리케이션 맵의 이름을 변경한다.

### a\) 서비스 → 어플리케이션 맵의 우측 ✎ 버튼을 클릭한다.
![]({{ site.baseurl }}/assets/KR/3.0.0/3.1.1-2_1.png)

### b\) 활성화된 이름 입력란에 내용을 변경 후, Enter 키를 눌러 어플리케이션 맵의 이름을 변경한다.
![]({{ site.baseurl }}/assets/KR/3.0.0/3.1.1-2_2.png)
