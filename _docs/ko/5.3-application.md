전체 어플리케이션의 총 CPU, Memory, Network 사용량과 개별 어플리케이션의 네임스페이스, 인스턴스 현황 등을 조회 할 수 있는 화면이다. 어플리케이션 명을 클릭하면 해당 어플리케이션 맵 페이지로 이동할 수 있다.

### a\) 클러스터 → 클러스터 선택 → 어플리케이션을 클릭한다.
