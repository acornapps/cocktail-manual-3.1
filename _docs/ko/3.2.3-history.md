해당 빌드의 히스토리 페이지로 이동하여 빌드 실행 항목에 대하여 히스토리 정보를 볼 수 있다. 빌드 실행 단위로 로그를 확인할 수 있다.

### a\) 서비스 → 빌드 관리 → 빌드 액션 메뉴\(활성화\) → 히스토리를 선택하여 해당 페이지로 이동
