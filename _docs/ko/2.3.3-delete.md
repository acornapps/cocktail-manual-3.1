서비스를 삭제 하는 기능이다. 삭제 시, 서비스에 클러스터가 등록되어 있을 경우 삭제가 불가능하며 서비스 편집에서 클러스터를 제거 후 삭제를 진행할 수 있다.

### a\) 환경설정 → 서비스 → 서비스 액션 메뉴 → 서비스 삭제를 클릭한다.
